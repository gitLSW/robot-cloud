Installation:

1. Install ROS Humble: https://docs.ros.org/en/humble/Installation/Ubuntu-Install-Debians.html
2. Install UR Description: sudo apt-get install ros-humble-ur-description
3. Install Omniverse: https://developer.nvidia.com/isaac-sim
4. In Omniverse install Isaac Sim
5. Launch Omnivere and enable ros2 bridge
6. Launch: ~/.local/share/ov/pkg/isaac_sim-2023.1.0/python.sh PATH/TO/CODE
7. Optinally set the Interpreter in VS to ~/.local/share/ov/pkg/isaac_sim-2023.1.0/python.sh
